<h1> WalletAPI </h1>

> Status: Em desenvolvimento, quase concluído.

## Conteúdo:
O projeto API RESTfull com Spring Boot utilizando TDD, CI e CD. Usando JUnit 4 e JPQL para parâmetros de busca, com autenticação por token.


# Tecnologias utilizadas
+ Java
+ Spring Boot
+ JPA / Hibernate
+ Bcrypt
+ JUnit 4
+ Maven
+ PostgreSQL
+ Postman
